# Quran Clone

A simple web application that lets users read, explore, and navigate the Holy Quran directly from the browser.

This project displays the complete text of the Quran with Surahs (chapters), Ayahs (verses), and tafsir (interpretation) pages, using local JSON data.

---

## 📌 Features

- Interactive Quran reader  
- Full list of all 114 Surahs  
- Individual verse display  
- Includes tafsir (extra interpretation) pages  
- Clean and responsive layout  
- Works offline (no server needed)
